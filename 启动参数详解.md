# 服务启动参数与环境变量详解

---

## 1. API (`packages/api/bin/api`)

### 命令行参数

| 参数 | 类型 | 必需 | 默认值 | 说明 |
|------|------|------|--------|------|
| `-port` | `int` | ❌ | `80` | HTTP 服务监听端口 |
| `-debug` | `string` | ❌ | `"false"` | 是否启用 Gin debug 模式。设为 `"true"` 时输出详细路由日志 |

**示例：**
```bash
./packages/api/bin/api --port 3000 --debug true
```

### 环境变量

#### 基础环境

| 变量 | 类型 | 必需 | 默认值 | 枚举值 | 说明 |
|------|------|------|--------|--------|------|
| `NODE_ID` | `string` | ✅ | - | - | 节点唯一标识，通常使用 hostname |
| `ENVIRONMENT` | `string` | ❌ | `"prod"` | `prod` `dev` `local` | 运行环境。<br>`prod`: 生产环境<br>`dev`: 开发环境<br>`local`: 本地开发，跳过等待延迟 |
| `E2B_DEBUG` | `string` | ❌ | `"false"` | `true` `false` | 启用 debug 级别日志 |

#### 数据库

| 变量 | 类型 | 必需 | 默认值 | 说明 |
|------|------|------|--------|------|
| `POSTGRES_CONNECTION_STRING` | `string` | ✅ | - | PostgreSQL 主库连接串，格式：`postgresql://user:pass@host:5432/db?sslmode=disable` |
| `DB_MAX_OPEN_CONNECTIONS` | `int32` | ❌ | `40` | DB 最大打开连接数 |
| `DB_MIN_IDLE_CONNECTIONS` | `int32` | ❌ | `5` | DB 最小空闲连接数 |
| `AUTH_DB_CONNECTION_STRING` | `string` | ❌ | 同 `POSTGRES_CONNECTION_STRING` | 认证数据库连接串（可独立配置） |
| `AUTH_DB_READ_REPLICA_CONNECTION_STRING` | `string` | ❌ | - | 认证数据库只读副本连接串 |
| `AUTH_DB_MAX_OPEN_CONNECTIONS` | `int32` | ❌ | `20` | 认证数据库最大连接数 |
| `AUTH_DB_MIN_IDLE_CONNECTIONS` | `int32` | ❌ | `5` | 认证数据库最小空闲连接 |

#### Redis

## 4. envd (`packages/envd/bin/envd`)

### 命令行参数

| 参数 | 类型 | 必需 | 默认值 | 枚举值 | 说明 |
|------|------|------|--------|--------|------|
| `-port` | `int64` | ❌ | `49983` | - | HTTP 服务监听端口 |
| `-version` | `bool` | ❌ | `false` | - | 打印版本号并退出 |
| `-commit` | `bool` | ❌ | `false` | - | 打印 Git commit SHA 并退出 |
| `-cmd` | `string` | ❌ | `""` | - | 启动时执行的 bash 命令（已废弃，推荐直接调用） |
| `-isnotfc` | `bool` | ❌ | `false` | - | 非 Firecracker 模式。为 `true` 时输出所有日志到 stdout |
| `-cgroup-root` | `string` | ❌ | `/sys/fs/cgroup` | - | cgroup v2 根目录路径 |

**示例：**
```bash
# Firecracker 模式（生产）
./packages/envd/bin/envd --port 49983

# 本地开发模式（非 Firecracker）
./packages/envd/bin/envd --port 49983 --isnotfc

# 查看版本
./packages/envd/bin/envd --version
```

### 环境变量

| 变量 | 类型 | 自动设置 | 默认值 | 说明 |
|------|------|----------|--------|------|
| `E2B_SANDBOX` | `string` | ✅ | `"true"`/`"false"` | 自动根据 `-isnotfc` 参数设置。`true` 表示运行在 sandbox 中 |

### cgroup 资源限制

envd 使用 cgroup v2 管理进程资源，默认配置：

| 进程类型 | cgroup 组 | cpu.weight | 内存限制 |
|----------|-----------|------------|----------|
| PTY 进程 | `ptys` | `200`（高优先级） | 总内存的 ~7/8 |
| Socat 进程 | `socats` | `150`（中优先级） | min=5MB, low=8MB |
| 用户进程 | `user` | `50`（低优先级） | 总内存的 ~7/8 |

## 通用环境变量

以下环境变量被所有服务共享：

| 变量 | 类型 | 必需 | 默认值 | 枚举值 | 说明 |
|------|------|------|--------|--------|------|
| `NODE_ID` | `string` | ✅ | - | - | 节点唯一标识（envd 除外） |
| `ENVIRONMENT` | `string` | ❌ | `"prod"` | `prod` `dev` `local` | 运行环境 |
| `E2B_DEBUG` | `string` | ❌ | `"false"` | `true` `false` | 启用 debug 日志 |
| `LOGS_COLLECTOR_ADDRESS` | `string` | ❌ | - | - | 日志收集器地址 |

### ENVIRONMENT 枚举值说明

| 值 | 说明 |
|----|------|
| `prod` | 生产环境。15 秒 drain 等待，使用 Nomad/KV 存储 |
| `dev` | 开发环境。同 `local` |
| `local` | 本地开发。跳过 drain 等待，使用本地存储 |

### STORAGE_PROVIDER 枚举值说明

| 值 | 说明 |
|----|------|
| `GCPBucket` | Google Cloud Storage（默认） |
| `AWSBucket` | Amazon S3 |
| `Local` | 本地文件系统 |

### VOLUME_TOKEN_SIGNING_METHOD 枚举值说明

| 值 | 密钥类型 | 说明 |
|----|----------|------|
| `ES256` | ECDSA P-256 | 椭圆曲线签名，推荐（体积小、性能好） |
| `RS256` | RSA 2048+ | RSA 签名，兼容性好 |
| `HS256` | HMAC | 对称密钥签名，简单但需安全分发 |
| `EdDSA` | Ed25519 | Edwards 曲线签名，最新最快 |
